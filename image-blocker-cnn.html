Image Blocker CNN

A Convolutional Neural Network for Image Filtering

View Code

Project Overview

This proof-of-concept project demonstrates the capability of Convolutional Neural Networks (CNNs) in image classification and filtering, specifically targeting the automated detection and blocking of unwanted images. Using dogs as a proxy for unwanted content, the project showcases how machine learning models can be applied to dynamically process and filter visual content in real-time.

CNN Algorithm & Model Training

The model leverages transfer learning with a MobileNetV2 base pre-trained on ImageNet. The process involves defining the network architecture, preprocessing image data, and training the model using a balanced dataset. The model achieved a 96.86% test accuracy.

Data & Preprocessing

To ensure the model learns generalized features, data was aggregated from three diverse sources: the 90 Animals Dataset, the Stanford Dogs Dataset, and the Mammals Image Classification Dataset. All images were resized to 128x128, normalized, and converted to NumPy arrays to create a perfectly balanced 50-50 dataset of 19,091 dog and 19,091 non-dog images.
