Project Overview

This custom compiler processes a C-like language (C-Lite) with support for variables, functions, arrays, control structures, and expressions. It performs lexical, syntax, and semantic analysis to transform high-level code into an intermediate representation (Three Address Code) and ultimately into executable MIPS assembly.

Supported Syntax

Variable Declarations

int x;
int y = 10;

Conditional Statements

if (x > y) {
    print(x);
} else {
    print(y);
}

Loops

while (i < 10) {
    i = i + 1;
}

Functions

int add(int a, int b) {
    return a + b;
}

Code Transformation

The compiler efficiently transforms C-Lite source code into platform-independent Three-Address Code (TAC) and then into MIPS assembly.

C-Lite Source

int x = 1 + 2 * 3 - 5;
print(x);

Generated TAC

t0 = 2 * 3
t1 = 1 + t0
t2 = t1 - 5
x = t2
print(x)

Generated MIPS

li $t0, 2
li $t1, 3
mul $t2, $t0, $t1
li $t3, 1
add $t4, $t3, $t2
li $t5, 5
sub $t6, $t4, $t5
sw $t6, x
lw $a0, x
li $v0, 1
syscall
