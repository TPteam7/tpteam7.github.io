TrueRead

An Image-to-Speech Application for Mathematical Equations

View Code

Project Overview

TrueRead is a desktop application that captures screenshots or uploads images to convert their content into spoken audio. It uses the MathPix API to process images containing mathematical content and OpenAI's ChatGPT to interpret equations in natural language, making it especially helpful for individuals with dyslexia.

Core Technologies

The application is built with a powerful stack of tools designed for user interaction, image processing, and AI-driven audio conversion.

PyQt5
Google Text-to-Speech
MathPix API
OpenAI (ChatGPT)
Pillow
Pygame

Installation & Configuration

To get started, clone the repository and install the required packages. You will also need to configure API keys for Google, MathPix, and OpenAI.

1. Install Packages

pip install PyQt5 Pillow pygame google-cloud-texttospeech requests

2. Clone Repository

git clone https://github.com/TPteam7/TrueRead.git
cd TrueRead

3. Set Environment Variables

export GOOGLE_APPLICATION_CREDENTIALS='path/to/creds.json'
export MATHPIX_APP_ID='your_mathpix_app_id'
export MATHPIX_API_KEY='your_mathpix_api_key'
export OPENAI_API_KEY='your_openai_api_key'

Usage

The interface provides simple controls to capture or upload an image and play the resulting audio.

  • Take a Screenshot: Click to select a portion of your screen to capture.
  • Upload an Image: Click to select an image file from your computer.
  • Play Audio: Once the image is processed, press to hear the audio representation.
